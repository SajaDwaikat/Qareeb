# Qareeb Mobile App

## Setup Instructions

1. Install dependencies:

```
npm install
```

2. Run the project:

```
npx expo start
```

3. Open on:

* Android emulator (press `a`)
* iOS simulator (press `i`)
* Expo Go

## Project Structure

* `(auth)` → Authentication screens
* `(tabs)` → Main navigation
* `property` → Property details
* `booking` → Booking flow

## Notes

* Use Expo Router
* Follow folder-based routing
